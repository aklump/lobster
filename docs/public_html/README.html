Getting started

If you are making an app called "Titanium"...

  1. Copy the folder called example_app as titanium somewhere on your computer and start building your app. That is the only folder you need from this repository to bundle with your app.
  2. Rename titanium/gitignore to titanium/.gitignore.
  3. Rename example_app.sh as titanium.sh; this will be the controller for the app.
  4. Copy dist/.lobsterconfig to titanium/.lobsterconfig and modify the necessary items.
  5. Copy dist/.example_appconfig to titanium/.titaniumconfig and use this file for any configuration used by your app.
  6. Sample routes have been provided in example_app/routes.

Documentation

  1. HTML documentation for Lobster is found at /docs/public_html.
