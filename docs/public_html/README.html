Index

A lightweight bash framework by In the Loft Studios.

When building a Lobster Shell app the core folder dist must be located, relative to the app's root at:

/lib/lobster/dist

Setup

If you are making an app called "Titanium"...

  1. In the root folder of your app, create a file called titanium.sh; this will be the controller for the app.
  2. In the root folder create the following folders: includes, routes, templates.
  3. copy the file dist/.lobsterconfig to your root folder and modify as needed.
  4. Refer to __lobster_app_example for how to build the app.
  5. Install lobster core at lib/lobster; you can rename gitignore as .gitignore for use in your app.

Documentation

  1. HTML documentation is found at /docs/public_html.
